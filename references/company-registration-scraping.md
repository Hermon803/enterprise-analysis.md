# 中国公司注册信息抓取实战

当企查查、天眼查、爱企查等企业数据库不可达（需要 JS/登录）时，用以下 curl 组合拳获取基本注册信息。

## 核心原则：双源交叉验证

| 数据源 | 能获取什么 | 局限性 |
|--------|-----------|--------|
| **公司官网** | 主营业务、产品线、资质荣誉、发展历程、联系方式 | 不展示注册资本/法人/信用代码等工商敏感信息 |
| **搜狗搜索摘要** | 注册资本、成立日期、法定代表人、统一社会信用代码、企业状态 | 依赖搜狗索引；非结构化，需从片段中提取 |

两条路径互不重叠，适合交叉验证。

## 路径一：官网直接抓取

```bash
# 公司名页面
curl -sL "https://www.<company>.com/about-us" \
  -H "User-Agent: Mozilla/5.0 (Win...Chrome/120)" 2>/dev/null

# 联系方式页面
curl -sL "https://www.<company>.com/contact-us" \
  -H "User-Agent: Mozilla/5.0 (Win...Chrome/120)" 2>/dev/null
```

从官网可提取：公司名称、电话、邮箱、地址、主营业务、产品线列表、资质荣誉、发展历程、ICP备案号。

### 获取ICP备案号

```bash
curl -sL "https://www.<company>.com/" | grep -oP '粤[^<]{3,30}'
```

ICP号可用于反查工商主体。

## 路径二：搜狗搜索抓注册信息

搜狗（sogou.com）是唯一对 curl 友好的中文搜索引擎，能返回包含企业注册信息的结构化摘要：

```bash
curl -sL "https://www.sogou.com/web?query=<URL编码的公司名>&ie=utf8" \
  -H "User-Agent: Mozilla/5.0 (Win...Chrome/120)" -m 10 \
  > /tmp/sogou_result.html
```

**提取中文文本的核心技巧**：排除所有 HTML 标签和 JS 噪声后，用 Unicode 范围匹配中文：

```bash
grep -oP '[\x{4e00}-\x{9fff}][\x{4e00}-\x{9fff}0-9\.\,\-\%\(\)\:\$\+/]{5,200}' \
  /tmp/sogou_result.html | sort -u
```

### 关键搜索词

| 目的 | 搜狗 query |
|------|-----------|
| 基本注册信息 | `"公司全名" 注册资本` |
| 信用代码 | `"公司全名" 统一社会信用代码` |
| 法人 | `"公司名" 法定代表人` |
| 搜索时务必用**引号包裹**公司全名 | |

### 搜狗摘要中的信息结构

搜狗本地商户/企业数据库展示的摘要格式通常是：

```
成立于<年份>, 位于<省份><城市>, ...企业注册资本<X>万人民币, 实缴资本<Y>万人民币。
<法人>经营下, 目前公司处于<状态>, 属于<行业>, 主营行业为<行业>
```

### 提取注册信息的具体方法

搜狗返回的 Sogou 搜索页中，企业的 **本地商户数据** 通常以内联文本片段方式嵌入在 HTML 中，而非 JS 动态渲染。

```bash
# 1. 搜索公司全名
curl -sL "https://www.sogou.com/web?query=%22%E5%85%AC%E5%8F%B8%E5%85%A8%E5%90%8D%22&ie=utf8" -H "..." > /tmp/sg.html

# 2. 提取所有中文文本块
grep -oP '[\x{4e00}-\x{9fff}][\x{4e00}-\x{9fff}0-9\.\,\-\%\(\)\:\$\+/]{5,200}' /tmp/sg.html | sort -u

# 3. 筛选注册相关条目
grep -E "注册资本|成立|法人|统一社会|实缴|纳税人|企业类型|经营状态|营业期限" /tmp/sg.txt
```

### 特殊技巧：搜狗不再返回摘要时的回退

```bash
# 搜索"公司名 注册资本"（不带引号但用空格分隔关键词）
curl -sL "https://www.sogou.com/web?query=公司名+注册资本" -H "..."

# 对超长公司名截断保留关键词（如只搜"深圳格芯"）
curl -sL "https://www.sogou.com/web?query=%22品牌名%22+%E6%B3%A8%E5%86%8C%E8%B5%84%E6%9C%AC" -H "..."
```

## 不可达的数据源（不要浪费时间尝试）

| 数据源 | 原因 |
|--------|------|
| 企查查 (qichacha.com) | 纯 JS 渲染，需要登录 |
| 天眼查 (tianyancha.com) | 同上 |
| 爱企查 (aiqicha.baidu.com) | 同上 + 反爬验证 |
| 国家企业信用信息公示系统 (gsxt.gov.cn) | JS 验证 + 反爬 |
| 百度 (baidu.com) | 安全验证码拦截 curl |

## 完整实战流程

```
Step 1: 官网扒取
  curl -> company.com/about-us
  提取: 主营业务、产品线、资质、发展历程、联系人信息

Step 2: 搜狗搜索
  curl -> sogou.com 带公司全名
  提取: 注册资本、成立日期、法人、实缴资本、企业状态

Step 3: 特殊数据抓取
  ICP备案号: grep 官网HTML底部
  统一社会信用代码: 搜狗搜索中 grep '9144|MA[0-9]'

Step 4: 交叉验证
  官网成立年份 vs 搜狗成立日期
  官网说的前身公司与企查查曾用名一致
```

## 已知局限性

1. **统一社会信用代码**：搜狗摘要偶尔展示，但不稳定。如搜到 `9144` 或 `MA` 开头的代码，用 `grep -oP '(9144|MA)[0-9A-Z]{0,20}'` 提取。注意搜狗可能返回同一页面中的其他公司代码，需人工判断。
2. **经营范围**：通常在官网简介中有提及，但完整条目仅在企业信用系统中有。搜狗摘要不会展示完整经营范围。
3. **股东/对外投资**：搜狗摘要不包含。需要企查查等专业数据库。
4. **注册资本变更历史**：搜狗只显示最新值。如有多个金额出现（如 3000万 vs 3333.3333万），标注为"可能存在注册资本变更"。

## 示例

参见 `格芯集成电路装备有限公司` 查询记录：2025-06-05 成功通过官网 + 搜狗组合获取全部七个维度的注册信息。
