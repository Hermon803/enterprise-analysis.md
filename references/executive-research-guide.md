# 创始人/核心管理层履历调研指南

用于企业六维分析中的"核心人物"维度。本文件记录核心人物数据的来源、数据项和搜索模式。

## 需要收集的数据项

| 数据项 | 示例 | 重要性 |
|--------|------|--------|
| 姓名 | 林宜龙、任正非 | 必填 |
| 籍贯/出生地 | 广东新会 | 必填 |
| 毕业院校及专业 | 广州轻工业学校 / 香港城市大学EMBA | 必填 |
| 关键履历（1-2句话） | "22岁创业，1995年创立格兰达，香港青年工业家奖" | 必填 |
| 当前职务 | 实际控制人、董事长、CEO | 必填 |

## 数据来源与优先级

| 优先级 | 来源 | 获取方式 | 可靠性 |
|--------|------|---------|--------|
| ① | 本地工作空间已有分析文件 | find ~/Progress/analysis/ | 高（已交叉验证） |
| ② | 百度百科 | 搜索"公司名 百度百科" | 高 |
| ③ | 公司官网"管理团队"页 | curl 官网 | 高 |
| ④ | 新三板/上市公司公告（董监高履历） | 证监会/股转系统披露 | 最高（法披信息） |
| ⑤ | 行业媒体报道（集微咨询、半导体行业观察等） | WebSearch | 中（需交叉验证） |
| ⑥ | 公开年鉴/人物志（如《江门人物志》） | WebSearch | 中 |

## 典型数据项示例

### 未上市民营企业（深圳格芯集成电路装备）

```
核心人物：林宜龙（实际控制人）
  - 籍贯：广东新会
  - 毕业院校：广州轻工业学校 / 香港城市大学EMBA
  - 履历：白手起家，22岁创业，1995年创立格兰达集团，香港青年工业家奖获得者
  - 法定代表人：周影绵（可查到工商注册信息）
```

### 大型上市/集团企业（华为）

```
核心人物：
  - 任正非（创始人）：贵州镇宁 · 重庆建筑工程学院 · 军队转业后创办华为
  - 孟晚舟（轮值董事长/CFO）：任正非之女 · 华中科技大学 · 华为财务体系奠基人
  - 梁华（董事长）：西北工业大学 · 从技术员到董事长
  - 余承东（终端BG董事长）：安徽霍邱 · 西北工业大学/清华大学 · 无线和终端业务领军
```

## 关键词搜索模板

```
百度搜索:     {公司名} 创始人
              {公司名} 实际控制人
              {公司名} CEO 背景
              {创始人姓名} 籍贯 毕业 履历
              {创始人姓名} 百度百科
工商信息:     {公司名} 法定代表人 天眼查/企查查
行业报道:     {创始人姓名} 集微咨询/半导体行业观察/36氪
```

## 常见问题

1. **法定代表人不等于创始人/实际控制人**：很多公司的法定代表人（如周影绵）是高管或挂名，实际控制人才是创始人（如林宜龙）。需要区分"工商登记上的法人"和"实际控制人"两个概念。

2. **未上市/非公众公司**：管理层履历披露较少，主要依赖百度百科、行业报道和公司官网。上市公司董监高履历在年报中必须详细披露。

3. **籍贯vs出生地**：有些人物籍贯和出生地不同，优先显示籍贯（祖籍）。如不确定，标注为"籍贯/出生地"并注明来源。

4. **毕业院校的时效性**：EMBA/MBA等后续学位可能晚于核心创业期，建议同时标注早年学历和后续进修经历。

5. **交叉验证**：不同来源的个人信息可能有冲突（如出生年份、早年履历），多源对比后取多数来源一致的版本，并在报告底部标注信源。

## 输出格式（六维报告中的标准行）

```
> **[姓名]** · [籍贯] · [毕业院校] · [关键履历一句话]
> **[姓名]**（[职务]）· [籍贯] · [毕业院校] · [关键履历一句话]
```

多核心人物时逐行列出。
