# 固定财务指标模板（10项 + 3年）

## 表格模板

每个公司统一使用以下 10 项指标，数据跨度为最近 3 年：

| 指标 | 2023年 | 2024年 | 2025年 | 类型 |
|------|--------|--------|--------|------|
| 营业总收入 | 填入 | 填入 | 填入 | 填写（上市必查） |
| 营收同比增速 | — | 自动 | 自动 | 计算 |
| 毛利率(%) | 填入 | 填入 | 填入 | 填写（上市必查） |
| 归母净利润 | 填入 | 填入 | 填入 | 填写（上市必查） |
| 净利率(%) | 自动 | 自动 | 自动 | 归母净利÷营收 |
| 经营活动现金流净额 | 填入 | 填入 | 填入 | 填写（上市可查） |
| 研发投入 | 填入 | 填入 | 填入 | 填写（上市可查） |
| 研发占营收比(%) | — | 自动 | 自动 | 研发÷营收 |
| 员工总数 | 填入 | 填入 | 填入 | 填写（所有企业） |
| 总资产 | 填入 | 填入 | 填入 | 填写（上市可查） |

## 数据来源分级

| 企业类型 | 营收/利润/毛利率/现金流 | 研发投入 | 员工 | 总资产 |
|----------|-----------------------|---------|------|--------|
| **A股上市** | 年报（巨潮资讯网/东方财富/同花顺） | 年报可查 | 年报可查 | 年报可查 |
| **非上市大企业** | 官网/新闻稿/搜狗搜索 | 部分可查 | ✅ 官网 | 部分可查 |
| **中小企业** | 搜狗注册信息（仅营收估算） | ❌ | ✅ 官网 | ❌ |

## 获取工具（A股上市）

Python (akshare):
```python
import akshare as ak
df = ak.stock_financial_abstract_ths(symbol='002261', indicator='按年度')
```

返回字段：报告期、净利润、净利润同比增长率、扣非净利润、营业总收入、营业总收入同比增长率、基本每股收益、每股净资产、每股经营现金流、销售净利率、销售毛利率、资产负债率。

## 不可获取时的处理

- 2023年研发投入若年报未单独列示：标注估算值并加 `*` 注释
- 非上市企业所有财务数据：标注"未公开披露"
- 经营现金流不可查：标注"未公开披露"
