# 两页PDF排版失衡调优技巧

## 现象

第二页仅包含产品展示区+数据源（约15%页高），导致视觉明显失衡。

## 错误直觉

缩小字号、边距、行距试图把所有内容"拉回"到页1。这会使页1密集拥挤，页2仍稀疏。

## 正确方法：反向调优

**增大字号+间距让页1自然溢出到页2**，同时独立放大页2元素：

### 步骤

1. **整体放大**：正文 8pt→9pt，行高 1.7→1.9，边距 1.3→1.5cm
2. **页2元素单独放大**：⑤⑥ blockquote 字号比页1大 0.5pt（9.5pt vs 9pt），产品图面积翻倍
3. **均衡目标**：中文字数比约 60:40（页:2）

### 示例参数

| 属性 | 紧凑版 | 均衡版 |
|------|--------|--------|
| 正文字号 | 8pt | 9-9.5pt |
| 行高 | 1.65 | 1.85-1.9 |
| bq 内边距 | 0.32em | 0.45-0.5em |
| 边距 | 1.3cm | 1.5cm |
| h1 字号 | 19pt | 20-21pt |
| 表格字号 | 7.5pt | 8.5-9pt |
| 产品图尺寸 | 80×55px | 90-110×62-75px |
