# 非上市/中小企业六维分析指南

本文件记录对非上市、未公开财报的中小企业（如"小巨人"、专精特新企业）进行分析的方法论。这类企业的数据获取路径与华为、百度等上市公司完全不同。

## 数据可获得性对比

| 数据类型 | 上市公司/巨头 | 非上市中小企业 |
|---------|-------------|---------------|
| 营收/利润 | 年报PDF精确到亿 | 无公开财务数据，最多从母公司公告/行业报告推算 |
| 员工数 | 财报披露精确人数 | 官网"关于我们"或招聘信息估算 |
| 税收贡献 | 年报中所得税可查 | 无数据 |
| 创始人履历 | 百度百科详尽 | 需多源拼凑（官网团队介绍、招聘网站、媒体报道） |
| 专利数据 | 国知局检索丰富 | 企查查/天眼查可查，但可能无PCT |
| 客户信息 | 年报披露大客户 | 官网"客户案例"页或行业报告 |
| 市场份额 | IDC/Omdia等有报告 | 集微咨询等细分行业报告（可能付费） |

## 非上市企业分析工作流差异

### Step 1 — 数据搜集重心转移

不再是"下载最新年报"，而是组合以下信源：

**优先级排列：**

| 优先级 | 信源 | 能获取的信息 |
|--------|------|------------|
| ① | **企业官网**（核心） | 主营业务、产品线、客户案例、团队介绍、荣誉资质 |
| ② | **百度百科** | 成立时间、注册资本、法定代表人、企业标签 |
| ③ | **企查查/天眼查**（如可达） | 工商信息、股东结构、专利列表、司法风险 |
| ④ | **母公司公开信息**（如有） | 如该企业是某集团子公司，查母公司年报/新三板公告 |
| ⑤ | **行业研究报告** | 集微咨询、36氪、亿欧等；提供市场地位和竞品信息 |
| ⑥ | **招聘平台** | 从岗位数量和招聘要求反推团队规模和技术方向 |
| ⑦ | **新闻/PR稿** | 企业动态、产品发布、融资信息 |

### Step 2 — 数据缺失时的处理规则

- **营收数据缺失** → 标注"未公开披露"，提供母公司/行业数据做参考
- **创始人照片缺失** → 尝试百度百科，仍无则标注"[图片待补充]"
- **市场份额缺失** → 使用定性描述（"国内领先""细分龙头"）替代定量数据
- **专利数据不完整** → 标注"据公开信息统计"，说明来源

### Step 3 — 图片下载特殊策略

对于非知名企业，SimpleIcons (`cdn.simpleicons.org`) 大概率不收录其Logo。替代路径：

```
Logo 抓取（非知名企业）:
  ① curl 官网首页 → grep -iE 'logo|icon|favicon' → 提取相对路径 → curl全URL
  ② 官网/RESOURCE/目录下找 logo.png / logo.jpg
  ③ 如果官网无独立Logo → 截图 或 纯文字输出

产品图抓取（非知名企业）:
  ① 官网产品/解决方案页 → grep -iE 'img.*jpg|img.*png' → 提取下载
  ② 中文科技媒体搜索"公司名+产品" → 评测/报道文章中的图片
```

### 实战案例：深圳格芯集成电路装备（2026-06-05）

**企业特征**：2020年成立、未上市、专精特新"小巨人"、母公司格兰达在新三板挂牌

**数据获取记录：**

| 信息 | 来源 | 可用性 |
|------|------|--------|
| 主营业务/产品线 | 官网 grandtec-ic.com (产品页完整) | ✅ 详尽 |
| 成立时间/注册资本 | 百度百科 / 官网 | ✅ |
| 专精特新"小巨人" | 官网荣誉资质页 | ✅ |
| 营收数据 | 母公司格兰达新三板挂牌公告（2024年10.42亿元） | ✅ 间接 |
| 员工~200人 | 官网发展历程页 | ✅ |
| 专利139项 | 官网+企查查（训练数据） | ⚠️ 数量级可靠 |
| 创始人林宜龙履历 | 百度百科+格兰达公告 | ✅ 交叉验证 |
| 创始人照片 | 百度百科 | ✅ 200x200 |
| Logo | 官网 `/resource/images/logo.png` (134x42 PNG) | ✅ |
| 产品图 | 官网产品页 `/uploads/images/...` | ✅ 260x160 PNG |
| 市场份额 | 集微咨询行业报告（三光机第一） | ✅ 定性+定量 |
| 核心客户 | 官网客户案例页（日月光/长电/华天等） | ✅ |

**关键教训**：
1. 非知名企业的官网通常比大公司官网信息更全（因为官网是其主要线上名片）
2. 母公司公开信息是最重要的间接数据源（格兰达新三板公告提供了营收等硬数据）
3. 集微咨询等垂直行业媒体对中小企业覆盖度好于IDC/Omdia等国际机构
4. logo.png/favicon 通常直接从官网首页HTML通过 `grep -iE 'icon|logo'` 提取
